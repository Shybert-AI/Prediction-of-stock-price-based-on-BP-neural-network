#  基于BP神经网络的股票价格预测
通过BP神经网络对明尼亚波利斯春小麦期货的收盘价进行了预测，准确的预测出收盘价的变动。  
BP神经网络中采用优化器ADAM、学习率变化策略Warmup和余弦退化技术，可以有效的解决训练时间太长的的问题。  
采用Dropout机制可以有效的提升模型预测的准确率和鲁棒性。
